# TodoList System
 
##这是一个简单的TodoList应用
 
使用了express作为web服务来启用
 
使用git下载项目并进入目录，执行命令  

**mongod** 启动本地mongodb数据库（前提：已经安装好mongodb）
**node ./bin/www** 来启动服务


##依赖介绍：

* Angular: [ 中文网站 ] (http://www.angularjs.cn)
* bootstrap: [ UI框架中文网站 ](http://www.bootcss.com)
* Moment: [ 日期格式化组件官网 ](http://momentjs.com)
